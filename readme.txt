== Description ==

BS23 Weather Widget

If anyone use that shortcode in any page, it render a widget, that shows current weather info.

Your shortcode should take params like location, temp unit ( C or F) etc.
 so, final shortcode will be like [show_weather unit="F" location="Dhaka" height="400px" width="400px"]
